# 19. API Documentation

The Streamlit app is the primary UI, but the system also exposes a thin **FastAPI**
service (optional, deployable alongside Streamlit) so that BI tools, ERPs and
other internal apps can consume forecasts programmatically.

> Base URL (prod): `https://api.sales-forecast.example.com/v1`
> All endpoints require `Authorization: Bearer <JWT>` unless stated otherwise.

---

## 19.1 Authentication

### `POST /auth/login`
Obtain a JWT.

**Request**
```json
{ "username": "ahmed", "password": "••••••••" }
```

**Response 200**
```json
{
  "access_token": "eyJhbGciOi...",
  "token_type":   "bearer",
  "expires_in":   3600,
  "role":         "business_user"
}
```

| Code | Meaning           |
|------|-------------------|
| 200  | OK                |
| 401  | Bad credentials   |
| 429  | Too many requests |

## 19.2 Forecast

### `POST /forecast`
Generate a forecast.

**Request**
```json
{
  "store_id":   12,
  "sku_id":     45,
  "horizon":    30,
  "scenario": {
    "promo":   true,
    "holiday": false
  }
}
```

**Response 200**
```json
{
  "model_version": "sales_xgb:9",
  "generated_at":  "2026-04-29T10:14:02Z",
  "horizon_days":  30,
  "forecast": [
    {"date": "2026-04-30", "yhat": 142.3, "yhat_lower": 130.1, "yhat_upper": 156.8},
    {"date": "2026-05-01", "yhat": 138.9, "yhat_lower": 126.4, "yhat_upper": 152.1}
  ],
  "metrics": {
    "expected_revenue": 84210.50,
    "expected_units":   4180
  }
}
```

| Code | Meaning                        |
|------|--------------------------------|
| 200  | Forecast generated             |
| 400  | Invalid params (validated by Pydantic) |
| 404  | Unknown store_id / sku_id       |
| 503  | Model registry unreachable      |

---

### `GET /forecast/batch?run_id={id}`
Returns the latest cached batch forecast as CSV (`Accept: text/csv`) or JSON.

---

## 19.3 Data

### `POST /data/upload`  (multipart/form-data)
Upload a sales CSV. Server validates schema, stores under DVC tracking.

**Form fields**
| name | type | required |
|------|------|----------|
| `file` | file (csv) | ✔ |
| `dataset_name` | string | ✔ |

**Response 201**
```json
{ "dataset": "sales_2026Q2", "rows": 412309, "dvc_hash": "md5:9f4...e2" }
```

### `GET /data/datasets`
List versioned datasets.

---

## 19.4 Models

### `GET /models`
```json
[
  {"name":"sales_xgb","version":"9","stage":"Production","mape":7.2},
  {"name":"sales_xgb","version":"8","stage":"Archived",  "mape":8.1}
]
```

### `POST /models/{name}/promote`
Body: `{ "version": 9, "stage": "Production" }` — **admin / ml_engineer only**.

---

## 19.5 Monitoring

### `GET /monitoring/drift?days=30`
```json
{
  "psi":   0.08,
  "mape":  7.2,
  "status":"ok"
}
```

### `POST /monitoring/retrain`  (admin / ml_engineer)
Triggers an asynchronous retraining job. Returns `202 Accepted` with `job_id`.

---

## 19.6 Standard error envelope

```json
{
  "error": {
    "code":    "VALIDATION_ERROR",
    "message": "horizon must be one of [7, 14, 30, 90]",
    "details": { "field": "horizon", "value": 45 }
  }
}
```

## 19.7 Rate limits
- 60 requests / minute / token (general)
- 5 requests / minute / token on `/forecast/batch` and `/monitoring/retrain`

## 19.8 OpenAPI
The full machine-readable spec is auto-generated by FastAPI at:
`GET /openapi.json` — Swagger UI at `/docs`, ReDoc at `/redoc`.
