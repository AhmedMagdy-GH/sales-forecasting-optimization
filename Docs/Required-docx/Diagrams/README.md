# 📐 Sales Forecasting & Optimization — Design Diagrams

This folder contains the full design documentation for the **Sales Forecasting & Optimization**
graduation project (IBM AI & Data Science Track — Project 2).

> **Tech stack:** Python · XGBoost · Pandas/NumPy · Scikit-learn · MLflow · DVC · Streamlit · Docker

All visual diagrams are written in **PlantUML** (`.puml`).
You can render them with:

```bash
# 1. Online
https://www.plantuml.com/plantuml/uml/

# 2. VS Code / PyCharm extension: "PlantUML"
# 3. CLI
java -jar plantuml.jar Docs/Diagrams/*.puml
```
